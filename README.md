# Echelon_Form

行列をとって、その階段形を掃き出し法による計算過程とともに返すプログラムです。階段形を求めるのは逆行列や一次方程式の解を求めるのに必要な作業ですが、手計算を行うのは面倒な一方、結果だけ見ても頭の中で行間を埋めるのは難しいです。そこで、手間なく計算過程を明示することを試みました。要はチート用です。

現状は(行数)>=(列数)なる行列にのみ対応しています。列数が上回る場合の0徐算の処理が直近の課題です。
